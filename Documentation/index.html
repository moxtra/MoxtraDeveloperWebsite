Getting Started

Unzip the archive and find out the folder www. There are all the template files in this folder. You can open the following files for viewing in browser.

  1. index.html — Home page with parallax image background.
  2. index-slider.html — Home page with full width slider.
  3. index-video.html — Home page with video background.
  4. index-pattern.html — Home page with pattern background.
  5. index-gradient.html — Home page with gradient background.
  6. shortcodes.html — the list of elements which you can use on your website.
  7. blog.html — Blog page.
  8. blog-post.html — Blog post.

Upload the template files (folder www) to server with the help of one of the FTP-clients (for example, Total Commander).

Note: for the correct operation of all functions of the template, including AJAX methods (portfolio expander), Twitter feed and Contact form, you must upload the template at web-server.

Note: Twitter feed customization guide you can find in section "Twitter Feed". Contact form customization guide you can find in section “Contacts Section”.

CSS Configuration file

To customize the stylistics of the template, open CSS configuration file: css/config.css

Fonts Combinations

You can choose one of the following font combinations, which are loaded from Google Web Fonts:

  1. Default: Montserrat & Lora.
  2. Open Sans (multi-language, Cyrillic supported).
  3. Istok Web & Noto Serif (multi-language, Cyrillic supported).

Find section "Fonts Combinations" in file css/config.css and uncomment one of the lines:

                        @import url("fonts-styles/fonts-1.css");
						
						/* @import url("fonts-styles/fonts-2.css"); */
						
						/* @import url("fonts-styles/fonts-3.css"); */

Main menu alignment

You can select one of the following options for the menu alignment:

  2. Bottom Right.
  3. Bottom Center.

  5. Top Right.
  6. Top Center.

Find section "Main Menu Alignment" in file css/config.css and uncomment one of the lines:

Logotype

To change logotype in navigation panel, find image images/logo-dark.png and replace it with your logo.

Note: to ensure Retina ready, the width and height of the image should be two times greater than the values specified in the attributes width и height. By default, the tag’s attributes logo size is set 86x27, but the actual size of the image is 172x54px.

To change logotype in Home Section, find image images/logo-white.png and replace it with your own logo.

Note: to ensure Retina ready, the width and height of the image should be two times greater than the values specified in the attributes width и height. By default, the tag’s attributes logo size is set 86x27, but the actual size of the image is 172x54px.

    3. Why I have problems with opening of works’ descriptions in portfolio while local browsing?
      Ajax methods of Jquery are used for loading descriptions of the works and they can work only on the real server. For the correct operation of project expander, upload your website on a real server.
    4. Why doesn’t Twitter feed work?
      For correct operation of Twitter feed, the php support is necessary. Upload your website on a server with php support.

      For the correct operation of contact form, the php support is necessary. Upload your website on a server with php support.
    6. Why there are no images that were in live demo?
      Envato forbids including demonstration images to templates’ files.
